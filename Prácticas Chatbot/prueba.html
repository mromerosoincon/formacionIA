Tareas por Semana

Semana 1: Configuración del Entorno y Fundamentos de Python

Día 1: Introducción, Configuración y Primeros Pasos

  • Instalar Python y Visual Studio Code.
  • Crear un archivo hola_mundo.py que imprima “Hola, Mundo” en la consola.
  • Ejecutar el script en VS Code y familiarizarse con el entorno.

Día 2: Variables y Operadores

  • Crear variables de diferentes tipos (int, float, str) y operar entre ellas.
  • Resolver ejercicios usando operadores aritméticos, de comparación y lógicos.
  • Practicar con operadores de asignación y explicar en comentarios cómo modifican las variables.

Día 3: Estructuras de Datos Básicas

  • Crear una lista, modificar sus elementos, agregar y eliminar valores.
  • Crear un diccionario que almacene información simple (por ejemplo, datos de un estudiante) y acceder a sus valores.
  • Comparar listas, tuplas, sets y diccionarios mediante ejemplos prácticos.

Día 4: Control de Flujo (Condicionales y Bucles)

  • Escribir un programa que solicite un valor numérico y determine si es positivo, negativo o cero.
  • Usar un bucle para recorrer una lista e imprimir cada elemento.
  • Realizar un ejercicio de “contador” usando un bucle while con una condición.

Día 5: Funciones, Clases y Manejo de Errores

  • Definir una función suma que reciba dos números como parámetros y devuelva la suma.
  • Crear una clase Persona con atributos nombre y edad, e inicializar un objeto.

Contenido Extra Opcional

  • Leer un archivo de texto e imprimir su contenido.
  • Usar expresiones regulares para buscar patrones en un texto (por ejemplo, encontrar correos electrónicos).
Práctica Final - Semana 1: Sistema de Calificaciones

Descripción del Problema: Crear un programa que gestione un sistema básico de calificaciones para un grupo de estudiantes.

Objetivo: Consolidar los conocimientos de Python aprendidos durante la semana.

Instrucciones de la Práctica:
  • 1. Crear un Diccionario de Estudiantes y Notas

    Crea un diccionario donde cada clave sea el nombre de un estudiante y cada valor sea su calificación (entre 0 y 10). Añade al menos cinco estudiantes con calificaciones diferentes.

  • 2. Calcular Promedios y Determinar el Estado de los Estudiantes
    • Promedio General: Calcula el promedio de todas las calificaciones.
    • Estado del Estudiante: Crea una función determinar_estado(calificacion) que devuelva "Aprobado" si la calificación es mayor o igual a 5, y "Reprobado" si es menor.
  • 3. Filtrar y Mostrar los Estudiantes Aprobados y Reprobados

    Crea dos listas para almacenar a los estudiantes aprobados y reprobados, respectivamente. Recorre el diccionario y clasifica a cada estudiante en la lista adecuada.

  • 4. Mostrar Resultados

    Imprime:

    • Los nombres y calificaciones de los estudiantes aprobados y reprobados.
    • El promedio general de calificaciones.
    • Un resumen que indique cuántos estudiantes aprobaron y cuántos reprobaron.

Ejemplo de Salida Esperada:

Estudiantes Aprobados:
- Ana: 7.5
- Luis: 6.0

Estudiantes Reprobados:
- Pedro: 4.5
- Marta: 3.0
- Juan: 2.5

Promedio General: 4.7
Resumen:
- Número de estudiantes aprobados: 2
- Número de estudiantes reprobados: 3
Pistas y Consejos:
  • Variables y Diccionarios: Usa variables para almacenar el promedio y listas para los estudiantes aprobados y reprobados.
  • Condicionales y Bucles: Utiliza un bucle for y condicionales if para determinar el estado de cada estudiante.
  • Funciones: Crea la función determinar_estado para simplificar el código.
Semana 2: Introducción a NLP y Modelos de Lenguaje

Día 1: Fundamentos de NLP y Vectores de Texto

  • Introducción a los vectores: investiga qué son y cómo representan el texto en NLP.
  • Bolsa de Palabras (Bag of Words): implementar un ejemplo simple en Python usando listas.
  • Ejercicio de Tokenización: Divide un texto en palabras individuales y elimina las "stop words".

Día 2: TF-IDF y Similitud de Vectores

  • Crear una representación de un texto usando el método TF-IDF.
  • Ejercicio de Similitud de Vectores: Calcula la similitud entre dos frases usando TF-IDF.
  • Proyecto Mini: Crea un recomendador de películas basado en similitud de textos usando TF-IDF.

Día 3: Embeddings de Palabras y Modelos Probabilísticos

  • Implementar embeddings de palabras usando Word2Vec.
  • Ejercicio de Analogías: Usar embeddings para resolver analogías en Python (por ejemplo, "Rey - Hombre + Mujer = Reina").
  • Modelos de Markov: Introducción y creación de una cadena de Markov simple en Python.

Día 4: Clasificación de Texto y Análisis de Sentimientos

  • Ejercicio de clasificación de texto usando un modelo Naive Bayes para detectar SPAM.
  • Implementación de un análisis de sentimientos básico usando regresión logística.
  • Crear un clasificador de sentimientos en Python con un pequeño conjunto de datos de texto.

Día 5: Resumen de Texto y Modelado de Temas

  • Ejercicio de Resumen de Textos: Implementa un resumen automático de un texto usando el método TextRank.
  • Modelado de temas con LDA (Latent Dirichlet Allocation): implementa una clasificación de temas en un conjunto de artículos.
Práctica Final - Semana 2: Clasificación de Textos y Análisis de Sentimientos

Descripción del Problema: Implementar un sistema de análisis de sentimientos y clasificación de texto utilizando un conjunto de datos de reseñas de productos.

Objetivo: Consolidar los conocimientos en procesamiento de lenguaje natural (NLP) y modelos de clasificación.

Instrucciones de la Práctica:
  • 1. Preparación del Dataset

    Recopila un conjunto de datos de reseñas o utiliza un dataset público. Divide el conjunto de datos en entrenamiento y prueba.

  • 2. Procesamiento de Texto
    • Tokenización: Divide las reseñas en palabras individuales.
    • Eliminar "stop words" y aplicar stemming o lematización.
  • 3. Representación y Clasificación
    • Vectoriza el texto usando TF-IDF o Word2Vec.
    • Entrena un clasificador Naive Bayes para la detección de sentimientos.
  • 4. Evaluación y Resultados

    Evalúa el rendimiento del modelo usando métricas como precisión, recall y F1-score. Muestra algunos ejemplos de clasificaciones correctas e incorrectas.

Ejemplo de Salida Esperada:

Reseñas Positivas:
- "Este producto es excelente, me encantó el diseño y la calidad."

Reseñas Negativas:
- "No recomiendo este producto, no cumple con lo que promete."

Precisión del modelo: 85%
Pistas y Consejos:
  • Preprocesamiento: Elimina caracteres especiales y normaliza el texto antes de vectorizarlo.
  • Modelo de Clasificación: Experimenta con diferentes modelos (Naive Bayes, regresión logística) para ver cuál da mejor resultado.
  • Métricas: Usa un conjunto de datos de prueba para evaluar el modelo y ajustarlo si es necesario.
